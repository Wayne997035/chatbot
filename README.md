# chatbot
### Line 天氣預報機器人
- 打openData API取得氣象資料
- 整理後判斷關鍵字地區，透過Line回覆


### 版本
- Java 11
- SpringBoot 2.5.4
- Jackson 2
- Gradle 7.2
- Junit 5
